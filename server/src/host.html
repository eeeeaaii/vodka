Quick Reference

 

Important note: some hotkeys are context-dependent.

Notably, inside a page nex, keys are reconfigured to allow you to type normally, as if you were using a word processor. So for example if you type "!", it will insert an exclamation point separator, not a boolean. You'll get the hang of it.

 

To see a list of commands natively built-in to Vodka, execute the "builtins" command.

 

To see a list of all bound symbols (such as library imports), execute the "bindings" command.

In command or symbol editor:

Any of a-zA-Z0-9:-_<>=+-/*can be used in names of commands or symbols.

deletedeletes previous character

option-spaceautocomplete

 

In lambda editor:

Any of a-zA-Z0-9_-can be used in names of function parameters.

deletedeletes previous character

n!n must be a boolean

n@n must evaluate to a symbol

n#n must be an integer

n$n must be a string

n%n must be a float

n^n must be a nil

n&n must evaluate to a closure

n*n must be an expectation

n()n must be some kind of nexcontainer

_nn will not be evaluated before passing to function (call by reference)

n...variable number of arguments (only valid as last parameter)

#%or %#either float or integer types allowed

?optional parameter

Multiple parameter specifiers can be combined, for example _n#%...

If no type is specified, any type is allowed.

 

In tag editor:

Any nex can be tagged. Any character including spaces can be used in a tag.

deletedeletes previous character

rightarrowor leftarrow

enterfinish editing tags and quit tag editor

go to next tag, or create new tag if no tag there (duplicate tags are deleted)

 

In string editor:

Type or paste the desired string in the box. Drag the handle to make the editor larger. All other hotkeys are disabled while you are editing the string. To finish editing the string, click the "done" button.
